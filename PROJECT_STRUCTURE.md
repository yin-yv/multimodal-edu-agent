# 项目结构说明

## 目录结构

```
multimodal-edu-agent/
├── backend/           # 后端服务
│   ├── core/          # C++ 核心
│   ├── agents/        # Python Agents
│   ├── services/      # 外部服务
│   └── api/           # API 接口
├── mobile/            # 移动应用
│   ├── flutter_app/   # Flutter 主应用
├── docs/              # 文档
├── scripts/           # 脚本
└── deployment/        # 部署配置
```

## 开发指南

1. 后端开发：进入 backend 目录
2. 移动端开发：进入 mobile/flutter_app 目录
3. 文档编写：进入 docs 目录
